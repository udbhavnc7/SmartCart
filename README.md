SmartCart AI is a modern, full-stack e-commerce application demonstrating advanced features like algorithmic bundle building, logistics optimization, and an integrated AI shopping assistant. 

### ✨ Key Features
- **Smart Bundle Builder:** Uses the knapsack algorithm to recommend the best products under a specific budget.
- **Advanced Logistics:** Features warehouse network simulation, route optimization, and smart packing logic.
- **AI Assistant:** Context-aware chatbot for product recommendations, budget picks, and user greetings.
- **Admin Dashboard:** Real-time analytics and full product CRUD operations.
- **Modern UX:** Dark/Light mode, Framer Motion animations, Recharts data visualization.

### 🛠️ Tech Stack
- **Frontend:** Next.js 16 (App Router), React, TypeScript, Tailwind CSS, Framer Motion, Recharts
- **Backend:** Express, Node.js, TypeScript, JWT Authentication
